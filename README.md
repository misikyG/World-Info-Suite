# World Info Suite

SillyTavern 擴充功能，提供增強的世界書 (World Info / Lorebook) 管理工具。

## 功能介紹

### 1. 觸發條目檢視器 (Triggered Entry Viewer)

檢視每則 AI 回覆觸發了哪些世界書條目。

- 在 AI 訊息上顯示地球圖示按鈕，點擊可查看該回覆觸發的所有世界書條目
- 依插入位置分組顯示（角色設定前/後、作者註釋前/後、深度插入等）
- 顯示條目來源（全域、主要知識、額外知識、聊天知識）
- 顯示觸發狀態（恆定、向量、關鍵字）
- 可查看主關鍵字、次要過濾器、內容等詳細資訊

**快取管理：**
- 自動限制快取數量，避免聊天記錄檔案過度膨脹
- 預設保留最近 10 則訊息的觸發資料
- 可在設定中調整保留數量（0 = 無限制）
- 提供一鍵清除所有快取的功能

### 2. 角色世界書快速跳轉 (Character Lorebook Quick Access)

快速存取角色綁定的世界書。

- 在角色面板中顯示綁定的世界書
- 區分主要世界書與額外世界書
- 點擊即可直接跳轉到對應的世界書

### 3. 批量條目編輯器 (Bulk Entry Editor)

一次批量編輯多個世界書條目。

- 在世界書編輯器中提供批量編輯按鈕
- 可選擇多個條目同時套用修改
- 支援修改所有條目屬性（關鍵字、內容、位置、深度等）
- 提供批量移動/複製條目到其他世界書的功能
- 提供批量刪除條目的功能
- 操作前會顯示確認對話框，列出所有將進行的變更

## 安裝方式

1. 下載或 clone 本擴充至 SillyTavern 的擴充資料夾：
   ```
   https://github.com/misikyG/World-Info-Suite
   ```
2. 重新啟動 SillyTavern
3. 在擴充功能設定中確認已啟用

## 設定選項

在 SillyTavern 的「擴充功能」設定面板中，可以：

- 啟用/停用各項功能
- 調整觸發檢視器的快取保留數量
- 清除所有快取的觸發資料

## 相容性

- 需要 SillyTavern 1.13.0 或更新版本
- 支援繁體中文與英文介面
